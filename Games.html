VR Baseball- "Shoot the Runner!"



  • Github Repo


In my Unity VR game, players find themselves on a baseball field faced with a wall of instructions as runners spawn. Armed with a gun, their task is to defeat these runners before they advance to the next base. Originally conceived as a batting simulator, then a catcher's perspective, the game evolved into a first-person shooter due to technical challenges. I'm proud of the unique grab interaction mechanic to trigger the game's start, and I have plans to enhance it with music, improved UI, and additional levels in the future, building on my excitement for game development.

CrackJack




  • Game File and Project Video


CrackJack is a Unity-based 2D platformer blackjack game with AR features. Despite challenges with Vuforia integration and UI design, the project provided valuable lessons in integrating diverse tools and collaborating effectively within a team, fostering innovation through shared idea generation and cooperation.

Guru Donuts Accesibility Compainion - AR



  • Github Repo


Our app helps those with non-verbal autism navigate places like Guru Donuts or Starbucks, focusing on streamlining the ordering process, which can be challenging due to social norms. We created a guide in the app to help users find lines and decide whether to wait or find a table. I contributed by designing the app's icon and image anchor for better accessibility and participated in user testing for the arrow initialization process within the establishments.
