Github



  • Visit PickTheStick


PickTheStick is a solo project inspired by attending live baseball games. Unlike traditional fantasy leagues, users select a single player per game to earn points based on performance. The long-term vision is a season-long competition where players cannot be selected more than once, with cumulative scoring on a global leaderboard.
